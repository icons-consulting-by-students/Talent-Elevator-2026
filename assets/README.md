# Medien-Upload Struktur

Bitte Dateien in diese Ordner legen:

- `assets/media/home` -> Hero-Bilder, Startseitenfotos
- `assets/media/events/2025` -> Fotos Event 2025
- `assets/media/events/2024` -> Fotos Event 2024
- `assets/media/events/2023` -> Fotos Event 2023
- `assets/media/about` -> Teamfotos, Portraits
- `assets/media/partners` -> Partnerbilder
- `assets/videos` -> Allgemeine Videos
- `assets/videos/2024` -> Eventvideo 2024
- `assets/logos` -> Firmenlogos

Empfohlene Formate:
- Bilder: `.jpg`, `.png`, `.webp`
- Logos: `.svg`, `.png`
- Videos: `.mp4`
